# Ecosystem Tooling

This folder highlights ecosystem partners and tools useful for building on Citrea (SDKs, infra, wallets, cross‑chain, data, and more) for developers.

* [**`interop-bridges-swaps.md`**](https://docs.citrea.xyz/developer-documentation/ecosystem-tooling/interop-bridges-swaps) covers cross‑chain messaging, third-party bridges, and intents‑based BTC swaps.
* [**`oracles-vrf.md`**](https://docs.citrea.xyz/developer-documentation/ecosystem-tooling/oracles-vrf) lists oracle providers and VRFs on Citrea with links to feeds and deployed contracts.
* [**`rpc-indexers.md`**](https://docs.citrea.xyz/developer-documentation/ecosystem-tooling/rpc-indexers) includes RPC access and indexing platforms for querying and mirroring on‑chain data.
* [**`utility-contracts.md`**](https://docs.citrea.xyz/developer-documentation/ecosystem-tooling/utility-contracts) lists utility contracts like Create2 Factory and CreateX deployed on Citrea.
* [**`wallets-aa.md`**](https://docs.citrea.xyz/developer-documentation/ecosystem-tooling/wallets-aa) details wallets and account‑abstraction tooling.

***

For the full list of partners, see the Citrea ecosystem page: [citrea.xyz/ecosystem](https://citrea.xyz/ecosystem).
